# iris.bot
WhatsApp Bot using Baileys
🌸 Iris Bot – Otaku WhatsApp Assistant

Bienvenue dans Iris Bot, un bot otaku conçu pour gérer et animer une communauté WhatsApp avec style et passion pour les mangas et animes.

🚀 Fonctionnalités principales

🤖 Réponses automatiques aux commandes (menu, règles, info)

🎌 Messages de bienvenue pour les nouveaux membres

📢 Diffusion d’annonces importantes dans le groupe

⚔️ Modération basique (anti-spam, rappel des règles)

🍙 Personnalisation otaku avec références à vos animes préférés

🖥 Installation

Clonez le projet :

git clone https://github.com/tonpseudo/iris-bot.git
cd iris-bot

Installez les dépendances :

npm install

Lancez le bot :

node src/index.js

Scannez le QR code avec WhatsApp pour connecter Iris.

📜 Commandes disponibles

menu → 🍙 Affiche le menu otaku

règles → ⚔️ Règles du groupe

info → 📖 Infos sur Iris

iris → 🌸 Présentation du bot

🍙 Menu Iris:
1️⃣ Règles

🌟 Thème Otaku

Iris est inspirée des univers manga et anime. Elle peut être personnalisée pour répondre avec des citations de vos séries préférées (Naruto, One Piece, Demon Slayer, etc.).

Les contributions sont les bienvenues ! Ajoutez vos propres commandes otaku et enrichissez Iris avec de nouvelles fonctionnalités.
